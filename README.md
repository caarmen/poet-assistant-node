# poet-assistant-node
Poet Assistant REST api using Node.js (with Express framework)

## Running
* Run the following commands:
  * `node app.js`
* Open the landing page at http://localhost:3000

## REST api documentation
https://caarmen.github.io/poet-assistant-node/
